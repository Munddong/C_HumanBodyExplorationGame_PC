# C_HumanBodyExplorationGame_PC
## #2. 인체를 탐험 하는 게임(PC)

## 1. 개발 계기

```
2018년 2학년 2학기 첫 PC게임 개발
게임제작기초 과목을 수강, 텀 프로젝트를 하기 위해 3명이 팀을 이루고 개발.
'게임'은 아이디어 싸움이라고 생각하여 탄생한 것이 인체를 탐험하는 게임. 
```

## 2. 역할

```
게임 기획 및 구상, 스크립트 작성과 시현 동영상 편집 등
```

## 3. 구현한 것

```
- 장기 모델링(Umodeler사용)
- HP UI
- 이벤트
- Etc
```

## 4. 키 조작

```
  ↑   : 앞
← ↓ → : 좌 뒤 우
Space : 아래
Left Ctrl : 위
```
